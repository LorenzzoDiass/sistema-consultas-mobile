# 🩺 Sistema de Consultas Mobile

Aplicação mobile desenvolvida com **React Native + TypeScript**, com foco na organização de código e boas práticas de desenvolvimento.

---

## 📌 Objetivo do Projeto

O objetivo deste projeto é simular um sistema de gerenciamento de consultas médicas, aplicando conceitos importantes como:

- Componentização
- Tipagem com TypeScript
- Separação de responsabilidades
- Organização escalável de projetos

---

## 🚀 Tecnologias Utilizadas

- React Native
- Expo
- TypeScript

---

## 📁 Estrutura do Projeto

O projeto foi organizado seguindo um padrão profissional:

src/
├── components/
│ └── ConsultaCard.tsx
│
├── screens/
│ └── Home.tsx
│
├── styles/
│ ├── app.styles.ts
│ └── consultaCard.styles.ts
│
├── interfaces/
│ ├── consulta.ts
│ └── medico.ts
│
└── types/
├── especialidade.ts
├── paciente.ts
└── statusConsulta.ts


---

## 🔧 Funcionalidades

- Exibição de consulta médica
- Visualização de médico e paciente
- Formatação de data e valor
- Confirmação de consulta
- Cancelamento de consulta
- Atualização dinâmica de status

---

## 🧠 Conceitos Aplicados

- Separação de responsabilidades (components, screens, styles)
- Uso de interfaces e types com TypeScript
- Organização escalável de projetos React Native
- Centralização de estilos
- Componentização reutilizável

---

## ▶️ Como Executar o Projeto

1. Instalar as dependências: npm install

2. Rodar o projeto: npx expo start

3. Abrir no navegador: pressione "w"
